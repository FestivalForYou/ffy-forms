## Synopsis

MySQL DB Module created by Festival For You to simplify query creation

## Dependencies

This module runs on top of the [mysql](https://www.npmjs.com/package/mysql) npm package created by Douglas Wilson.

## Code Example

```
var db = require('ffy-db');
var options = {
  host: '127.0.0.1',
  user: 'root',
  password: '',
  database: 'database'
}

db.setup(options);
```
